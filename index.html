🚀 Getting Started

Quick Start Guide

  1. Upload Resume: Select a candidate's resume file (PDF, DOCX, or TXT)
  2. Add Job Description: Either upload a job description file or paste the text
  3. Launch Evaluation: Click the evaluation button to start AI analysis
  4. Review Results: Analyze the comprehensive assessment report
  5. Make Decision: Use the insights to make informed hiring decisions

📚 Feature Guide

📋 Single Candidate Evaluation

Evaluate individual candidates with detailed skill matching and recommendations.

  • Resume parsing and analysis
  • Skills gap identification
  • Experience level assessment
  • HR decision support

📚 Bulk Screening

Process multiple resumes simultaneously for efficient batch evaluation.

  • Batch processing (up to 20 files)
  • Comparative analysis

🔧 Troubleshooting

Common Issues & Solutions

❌ File Upload Issues
  • Ensure file size is under 10MB
  • Use supported formats: PDF, DOCX, TXT
  • Check file permissions and try again
❌ Analysis Not Working
  • Verify both resume and job description are provided
  • Check internet connection
  • Try refreshing the page
